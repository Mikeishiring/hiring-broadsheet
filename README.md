# Hiring Intelligence — A Broadsheet Blog

A knowledge base on hiring craft, built with the [Broadsheet](https://github.com/Mikeishiring/broadsheet) layout engine.

Three long-form essays explore the structural problems in how organizations evaluate talent:

- **The Signal Problem** — how most hiring processes optimize for the wrong signals, measuring confidence instead of competence
- **Interview Architecture** — designing interview processes as instruments of perception, not bureaucratic checkboxes
- **The Calibration Gap** — why interviewers are rarely calibrated against outcomes and what it costs

## Running locally

```bash
npm install
npm run dev
```

## Features

- Two-column editorial layout with live text wrapping around draggable obstacles
- Parchment and dark editorial themes
- Read, edit, and arrange modes — edit copy directly in the layout, drag obstacles to reshape the composition
- Responsive typography with binary-search headline fitting
- Custom SVG obstacle shapes themed to each essay
